1. Project Goals

This project aims to guide beginners in building their first Flask web application ‚Äî a simple yet fully functional student sign-in system. It demonstrates how Flask handles routing, templates, and session management in a clean and modular way, helping students transition from Python basics to full-stack web development.

2. Project Workflow

The application follows a simple yet clear workflow from user interaction to server response:

  1. The user opens the website in their browser.
  2. Flask handles the HTTP request and matches it to the correct route (such as /home or /tutorial).
  3. The route calls its associated view function, which loads and renders an HTML template using Jinja2.
  4. Static files (CSS, JavaScript, and images) are applied to style and structure the page.
  5. The fully rendered HTML page is returned to the user‚Äôs browser.
